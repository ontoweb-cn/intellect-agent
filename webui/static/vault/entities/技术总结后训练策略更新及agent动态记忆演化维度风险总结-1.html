一、后训练策略更新技术总结

来看一个技术总结，以轨迹来源为核心维度，将大模型后训练，按照轨迹来源分类，可以分为离策略学习【基于外部提供轨迹更新模型】、在线策略学习【基于当前模型生成轨迹更新模型】 ，工作在《Large Language Model Post-Training: A Unified View of Off-Policy and On-Policy Learning 》(https://arxiv.org/pdf/2604.07941)，可以看几个核心点。

来看文中表达的几个洞察：

1、后训练本质是对模型行为的结构化干预 ，所有后训练方法（SFT、偏好优化、RL、蒸馏等）最终都在改变模型的轨迹分布 ，而非仅仅拟合标签或优化单一目标。其学习信号包括Token、偏好、奖励、验证器、教师指导 等。

2、任何后训练阶段的行为改变 ，都可归为三类作用：有效支持扩展 ：让模型能生成之前达不到的有用行为、策略重塑 ：在已有可达行为内优化概率分配、行为巩固 ：跨阶段/模型保留、迁移、压缩有效行为。

3、离线策略方法（SFT、离线偏好优化、离线蒸馏 ）擅长有效支持扩展与离线校准，但无法直接修正模型自生状态下的错误。

4、在线策略方法（RLHF、RLVR、过程监督 ）擅长策略重塑，能直接优化模型实际生成时的中间状态与多步错误 ，但难以自主引入全新外部行为。

5、SFT既可做支持扩展（引入新推理结构） ，也可做策略重塑（格式、风格、对齐校准 ），标准离线偏好优化几乎不扩展新行为，只在模型已有可达输出中做排序与对齐校准。

6、大模型蒸馏的主要价值是跨阶段保留、迁移、摊销昂贵行为（搜索、RL、多步推理） ，而非单纯缩小模型。

7、后训练存在多阶段瓶颈 ：典型流水线模式包括SFT→偏好优化 ：先建立行为，再校准排序与对齐；SFT→RLHF/RLVR ：先引入行为，再修正模型自生状态错误；SFT→RL→蒸馏 ：先引入→再优化→最后巩固压缩部署。

但是这都是先需要引入行为（扩展），再需要修正自生错误（重塑），最后需要稳定部署（巩固），单一范式无法全覆盖。

二、Agent动态记忆演化维度及风险

看第二个话题，早期LLM智能体为无状态，依赖固定上下文窗口与静态RAG，无法自主学习与更新，所以，现在都开始往记忆更新的方向走，可以看个技术总结，把握下方向，在《Governing Evolving Memory in LLM Agents: Risks, Mechanisms, and the Stability and Safety Governed Memory (SSGM) Framework 》(https://arxiv.org/pdf/2603.11768)。

核心还是看几个点：

1、动态记忆的几个演化维度

从典型系统MemGPT、Memory-R1、A-MEM、HiMem、AtomMem等，从中，可以归纳出几个方向：

内容方向，包括添加/更新/摘要/删除，支持多模态与过程性知识;

结构方向，从向量库→知识图谱→层次图/自组织图；

策略方向，反思更新、强化学习优化、记忆巩固与遗忘

2、动态记忆核心风险

风险可以进一步的归类，包括：

稳定性失效 ，包括语义漂移【由迭代摘要、有损压缩导致，表现为事实逐步失真、用户偏好被过度强化】；过程漂移【因固化次优策略引发，表现为智能体执行路径出现退化】；记忆幻觉【由错误内容内化造成，表现为检索时输出虚假信息】；

有效性失效 ，包括知识过时【因未设置时间衰减机制导致，表现为智能体基于过期数据做出决策】；

效率失效 ，包括延迟膨胀【由记忆体量线性增长引发，表现为检索速度呈指数级下降】；

安全性失效 ，包括记忆污染【因恶意指令被注入导致，表现为智能体行为被恶意操控】、隐私泄露【由记忆拓扑结构存在缺陷引发，表现为跨会话、跨用户的数据泄露】
